Chocolate Shipment & Sales Analysis:
This project analyzes chocolate shipment and sales data to generate business insights using Power BI. The dashboard helps track sales performance, shipment status, product demand, and regional trends.

 #Tools Used
* Power BI
* Excel / CSV Dataset
* DAX
* Power Query

 #Dashboard Features
* Total Sales & Profit KPIs
* Shipment Status Analysis
* Product-wise Sales Performance
* Region-wise Sales Comparison
* Monthly Sales Trend
* Interactive Filters

 #Key Insights
* Identified top-selling products
* Analyzed high and low performing regions
* Detected shipment delays
* Observed seasonal sales trends
